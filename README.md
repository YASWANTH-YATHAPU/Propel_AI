# PropelPro AI — Sales Proposal Architect

PropelPro AI is a browser-based sales proposal generator that helps users create professional, customized client proposals with the assistance of Google Gemini. It provides an intuitive form-driven interface for entering client details, project scope, budget, timeline, and proposal preferences, then generates a polished proposal in seconds.

## Preview

A clean, responsive two-panel interface:
- Left panel: proposal inputs and configuration
- Right panel: generated output, status updates, and action buttons

## Features

- AI-powered proposal generation using Google Gemini
- Client and company information input
- Industry and company size selection
- Project scope and pain-point description
- Service selection using interactive chips
- Budget slider with live currency display
- Project duration and tone selection
- Section-based proposal generation
- Executive Summary regeneration
- Risks & Assumptions regeneration
- Version comparison between proposal drafts
- Copy proposal text to clipboard
- Responsive design for smaller screens

## Tech Stack

- **HTML5** — page structure
- **CSS3** — styling and responsive layout
- **JavaScript (ES6)** — UI interactions and API integration
- **Google Gemini API** — text generation
